# 🧮 Ultimate Calculator in C

A menu-driven calculator built in **C** that goes beyond basic arithmetic. This project is designed to combine common mathematical operations, number utilities, and conversion tools into one simple console application.

> 🚧 This project is currently under development. New features and improvements are being added regularly.

---

## ✨ Features

### 🧮 Basic Calculator
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division

### 🔢 Number Utilities *(In Progress)*
- Reverse a Number
- Count Digits
- Sum of Digits
- Greatest Common Divisor (GCD)
- Lowest Common Multiple (LCM)

### 📚 Mathematical Programs *(Planned)*
- Multiplication Table
- Sum of First N Natural Numbers
- Sum of Odd Numbers
- Sum of Even Numbers

---

## 🎯 Vision

The goal of this project is to create an **all-in-one mathematical toolkit** using the C programming language. Instead of writing separate programs for every concept, this application brings them together into a single interactive menu-driven program.

Future updates will include:
- Prime Number Checker
- Palindrome Checker
- Armstrong Number Checker
- Perfect Number Checker
- Fibonacci Series
- Factorial
- Square, Cube & Power
- Square Root
- Average, Mean & Median
- Percentage Calculator
- Temperature Conversion
- Distance Conversion
- Binary ↔ Decimal Conversion
- And many more...

## ▶️ How to Run

Compile:

```bash
gcc main.c -o main
```

Run:

```bash
./main
```

On Windows:

```bash
main.exe
```
